# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly by contacting the owner directly:

**Aryan Rajendra Suthar**
aryanrajendrasuthar@gmail.com

Please do NOT open a public GitHub issue for security vulnerabilities.

## Response

The owner will acknowledge receipt within 72 hours and provide a resolution
timeline. Responsible disclosure is appreciated.

## Scope

This policy applies to the AI Knowledge Assistant codebase and all associated
services under this repository.
